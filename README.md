# ⚡ Proyecto destacado: Z-Commerce

Desarrollé y desplegué **Z-Commerce**, una plataforma e-commerce full-stack construida con **ASP.NET Core MVC**, **PostgreSQL**, **Entity Framework Core**, **ASP.NET Core Identity**, **Docker** y despliegue en la nube mediante **Render**.

🌐 **Demo en vivo:**
https://z-commerce-dh8r.onrender.com/

📦 **Repositorio:**
https://github.com/SebastianASR/Z-Commerce

## Funcionalidades principales

* Autenticación de usuarios con ASP.NET Core Identity.
* Registro e inicio de sesión con roles `Admin`, `DemoAdmin` y `Cliente`.
* Política de contraseñas robusta.
* Bloqueo temporal por intentos fallidos de inicio de sesión.
* Confirmación real de correo electrónico.
* Recuperación de contraseña mediante Brevo API.
* Panel administrativo para gestión de productos, pedidos y solicitudes.
* CRUD completo de productos.
* Catálogo de productos.
* Carrito de compras con sesiones.
* Checkout funcional para usuario registrado e invitado.
* Integración con Transbank Webpay Plus Sandbox.
* Base de datos PostgreSQL alojada en Neon.
* Despliegue real con Docker + Render.
* Configuración segura mediante variables de entorno.
* Uso de `appsettings.Example.json` para evitar exposición de credenciales.
* Usuario DemoAdmin limitado para pruebas seguras en producción.

Este proyecto representa mi avance desde un CRUD tradicional hacia una aplicación web más completa, con autenticación, persistencia real, despliegue cloud, integración de APIs externas y buenas prácticas de seguridad.

# -> Proyecto en desarrollo: Z-Entik

Actualmente estoy desarrollando **Z-Entik**, un sistema HelpDesk TI para gestión de tickets, usuarios, roles, técnicos, estados, prioridades y comentarios.

📦 **Repositorio:**
https://github.com/SebastianASR/Z-Entik

## Stack planificado

* React + TypeScript
* Vite
* NestJS
* Node.js
* PostgreSQL
* Prisma ORM
* JWT
* Hash seguro de contraseñas
* 2FA por correo
* Brevo API
* Docker
* GitHub Actions
* Render
* Neon PostgreSQL

## Objetivo del proyecto

Z-Entik busca demostrar una arquitectura Full Stack moderna, separando frontend y backend mediante una API REST, integrando autenticación segura, base de datos cloud, despliegue e infraestructura básica de CI/CD.
